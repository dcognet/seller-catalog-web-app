# Seller Catalog Web App

## Getting Started

Create a .env file

```shell
VITE_API_URL=http://localhost:3001
```

Install the dependencies

```shell
yarn install
```

Run the dev server

```shell
yarn dev
```

Commit using

```shell
yarn commit
```
